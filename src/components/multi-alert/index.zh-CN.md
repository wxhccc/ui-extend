### Attributes

| 参数          | 类型              | 说明                         | 可选值 | 默认值 |
| :------------ | :---------------- | :--------------------------- | :----- | :----- |
| data          | object[]          | 数据对象数组                 | --     | --     |
| └ title       | string            | 标题                         | --     | --     |
| └ description | string / string[] | 描述内容字符串或列表内容数组 | --     | --     |
